Mario Durango

## Descripción

Este proyecto consiste en el desarrollo de un dashboard interactivo utilizando Python y Streamlit. El objetivo es visualizar datos relacionados con estudiantes universitarios de manera clara y comprensible.

## Dataset

Se utilizó el archivo `university_student_data.csv`, el cual contiene información sobre:

* Año académico
* Periodo (Spring/Fall)
* Número de estudiantes matriculados
* Tasa de retención
* Nivel de satisfacción estudiantil

## Visualizaciones

El dashboard incluye:

* Tendencia de la tasa de retención a lo largo de los años
* Nivel de satisfacción estudiantil por año
* Comparación entre los periodos académicos
* Filtros interactivos por año y periodo

## Herramientas utilizadas

* Python
* Streamlit
* Pandas
* Matplotlib
* Seaborn

## Ejecución

Para ejecutar el proyecto:
streamlit run app.py

## Conclusión

El dashboard permite analizar los datos de forma visual e interactiva, facilitando la comprensión de la información y apoyando la toma de decisiones.
